## Commands
```
/{TOPIC} {SUBTOPIC}
/help
/topics - list all searchable topics
/add - add topic to commands
/python
/rust
/linux
/bash
/git

Everytime you request the same subtopic it will respond with a new cheat sheet variation of that subtopic.
```
